## 数组

PHP 中的数组实际上是一个有序映射 .映射是一种把values 关联到key 的类型。

### **定义数组 array\(\)**

* key 可以是整数 integer 或字符串 string 
* 值 value 可以是任意类型的值

> 最后一个数组单元之后的逗号可以省略。通常用于单行数组定义中，例如常用 _array\(1, 2\) _而不是 _array\(1, 2, \)，_对多行数组定义通常保留最后一个逗号，这样要添加一个新单元时更方便。

自 5.4 起可以使用短数组定义语法，用\[\]来定义数组。

```
$array = array(
    "foo" => "bar",
    "bar" => "foo",
);

$array = [
    "foo" => "bar",
    "bar" => "foo",
];
```
